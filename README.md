# GoodReason
Specification for GoodReason - worldview for human and AI understanding knowledge

GRN.1 – GoodReason Notation v1.0
Minispesifikaation runko

Johdanto

GoodReason Notation (GRN.1) määrittelee universaalin symbolisen tietoyksikön kahdeksan sektorin (α–Ω) ja seitsemän tason (1–7) perusteella.
GRN.1 toimii:

- käyttäjälle mallirakennusvinkkien ja kontekstuaalisen esimerkkisanaston lähteenä

- tekoälylle symbol grounding -kerroksena, jonka avulla AI voi tulkita kyselyä ja ohjata dialogia

- ohjelmistoille (Prolog, web) formaalisena tietorakenteena (JSON + Prolog + UI)

GRN.1 on tarkoitettu yhden tiedon atomin formaatiksi: yhden sektoritason merkitysyksiköksi.
